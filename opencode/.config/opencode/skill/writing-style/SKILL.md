# Writing Style

- Act as editor, not author. Keep edits small unless asked otherwise. Preserve original voice and structure.
- Imperative mood, American English.
- Lead with problem or context before solution.
- Use "we" for collaboration; "you" to address the reader.
- Paragraphs: 2-4 sentences max.
- Explain the why, not just the what.
- Link liberally to sources, docs, references.
- Bullet points over numbered lists unless order matters.
- Direct and opinionated; acknowledge tradeoffs honestly.
- Bold key phrases that anchor an argument.
- Rhetorical questions sparingly.
- Prefer AP style unless the project has an existing convention.
- Em-dashes OK. Semi-colons less so.
- No emojis unless user uses them.
- No marketing speak: "perfect for", "empowers you to", "modernization".
